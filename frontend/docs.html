API Documentation

Welcome to the OpenMediaFlow API. This guide explains how to use our Media generation REST API.

Authentication

All requests must include an API key provided in the x-api-key header. 

POST /api/reference/generate-image/:id
x-api-key: YOUR_API_KEY

Get your API Key

Base URL

All endpoints are relative to:

https://media-ai-backend.vercel.app

Endpoints

POST /api/reference/generate-image/:modelid

Generates image for the specified model ID.

Request Body:

{
  "prompt": "a red panda eating a bamboo in front of a poster that says \"recraft V3 now available at fal\""
}

Example cURL Request:

curl -X POST -H "x-api-key: YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "prompt": "a red panda eating a bamboo in front of a poster that says \"recraft V3 now available at fal\""
  }' \
  https://media-ai-backend.vercel.app/api/reference/generate-image/black-forest-labs-flux-1-1-pro
POST /api/reference/generate-video/:id

Generates audio for the specified resource ID.

Request Body:

{
  "prompt": "Two anthropomorphic cats in comfy boxing gear and bright gloves fight intensely on a spotlighted stage."
}

Example cURL Request:

curl -X POST -H "x-api-key: YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "prompt": "Two anthropomorphic cats in comfy boxing gear and bright gloves fight intensely on a spotlighted stage."
  }' \
  https://media-ai-backend.vercel.app/api/reference/generate-video/wan-ai-wan21-t2v-13b
POST /api/reference/generate-audio/:id

Generates audio for the specified resource ID.

Request Body:

{
  "prompt": "dog barking in rain",
  "duration": 10
}

Example cURL Request:

curl -X POST -H "x-api-key: YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "prompt": "dog barking in rain",
    "duration": 10
  }' \
  https://media-ai-backend.vercel.app/api/reference/generate-audio/stackadoc-stable-audio
